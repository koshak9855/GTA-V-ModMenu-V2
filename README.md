## 📋 Features

- **🛡️ External Execution:** Operates entirely outside the game's memory space, providing enhanced stability and process safety.
- **📟 User Interface:** Low-latency Windows GDI+ / DirectX 11 overlay for real-time interaction.
- **📜 Script Integration:** Built-in engine to load and execute custom `.lua` or external logic scripts.
- **🖥️ Hardware Optimization:** Compiled using WinAPI/C++ to ensure 0% CPU overhead during intensive gaming sessions.
